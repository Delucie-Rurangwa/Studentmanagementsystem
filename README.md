Reflection Questions


1.What is the difference between a class and an object?   

Answer: -A class is a conceptual blueprint or template that defines variables and methods.
        -An object is a concrete instance created from that class that actually holds real data values in memory.  
        
2.Why do we use constructors in Java?  

Answer: We use constructors to initialize the attributes of an object automatically the moment it is created using the new keyword, ensuring the object starts with valid data.

3.What is the difference between static and non-static methods?   

Answer: -A static method belongs to the class itself and can be called directly using the class name without creating an object.

        -A non-static (instance) method belongs to individual objects and requires an object to be instantiated before it can be called.  
        
4.What is variable scope?   

Answer: Variable scope refers to the specific region of a program's code (usually bounded by curly braces { }) where a variable is defined, recognized, and accessible. 

5.What happens when a local variable is accessed outside its scope? 

Answer: It causes a compilation error because the variable ceases to exist in memory once the program exits the block of code where that variable was declared.
